# MaYa Raider

Welcome to MaYa Raider game, a Java and JavaFX implementation of a strategic game inspired by Clash of Clans. This project was developed as part of an Advanced Programming course at University of Isfahan.

## Installation

To run MaYa Raider, follow these steps:

1. Clone the repository.
2. Open the project in your preferred Java development environment.
3. Build and run the project.
